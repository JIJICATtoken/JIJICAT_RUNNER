# 👑 JIJICAT RUNNER

**PLAY • EARN • DOMINATE**

A futuristic 3D endless runner game featuring a royal white glowing-eyed cat with a gold crown in a neon crypto-themed environment.

## 🎮 Features

- ⚡ **Endless Runner Gameplay** - Auto-forward movement with progressive difficulty
- 🎨 **Neon + Crypto Theme** - Futuristic grid background with glowing effects
- 🐱 **Royal Cat Character** - White cat with gold crown and glowing eyes
- 🎯 **Smooth Controls** - Swipe controls for mobile, keyboard for desktop
- 💰 **Collectible System** - Gold coins with particle effects
- 🛡️ **Power-ups** - Magnet, Shield, Speed Boost
- 🔥 **Multiple Obstacles** - Fire blocks, laser barriers, platform gaps
- 📊 **Combo Multiplier** - Score increases with consecutive coin collection
- 🏆 **Leaderboard** - API-ready for Supabase integration
- 📱 **Responsive Design** - Optimized for mobile and Telegram Mini App

## 🕹️ Controls

- **Jump**: Arrow Up / W / Swipe Up
- **Slide**: Arrow Down / S / Swipe Down
- **Move Left**: Arrow Left / A / Swipe Left
- **Move Right**: Arrow Right / D / Swipe Right
